# TODO вЂ” task history

Append-only log of non-trivial tasks and their status. Add a task when you start
it, flip it to **In progress**, then **Done** when finished вЂ” never delete
completed entries (this is the history). Each line: `YYYY-MM-DD вЂ” description`.
Status markers: `- [ ]` pending В· `- [~]` in progress В· `- [x]` done В·
`- [-]` cancelled/blocked (note why). Newest first within each group.
