## 0.1.0

Initial scaffold.

- JupyterLab 4.x prebuilt extension with an Airflow DAG sidebar panel
  (list, pause/unpause, trigger, recent runs).
- Jupyter server extension proxying the Apache Airflow 3.x REST API (`/api/v2`)
  with JWT auth via `/auth/token`.
- Devcontainer with a local Apache Airflow 3.x (standalone) service.
